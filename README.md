## Overview

The Peer Learning Platform is a web application designed to help students learn from one another through skill sharing and tutoring sessions. The platform creates a collaborative learning environment where users can teach skills they are confident in and book sessions to learn new topics from their peers.

## Description

The Peer Learning Platform is a web-based application that allows students to connect, share knowledge, and learn from one another through peer tutoring sessions. The platform enables users to create accounts, post skills they can teach, and book learning sessions with other students.

The main goal of the project is to encourage collaborative learning by creating an easy-to-use environment where students can exchange knowledge and improve their academic or technical skills. The application includes user authentication, a student dashboard, skill posting, and session booking features.

The project was built using React for the frontend and Firebase for authentication and database storage. It also uses React Router for navigation and Tailwind CSS for responsive and modern user interface design. The platform is designed to provide a smooth and interactive user experience across different devices.
